# Proyecto-I---Catalogo-de-Superheroes
Software de gestión para el catálogo virtual de la tienda Mundo Comic. Este sistema fue desarrollado en Python con PyQt6. Permite administrar el catálogo de historietas de Marvel y la información de sus personajes. Se implementaron estructuras de datos lineales (listas simples, dobles y circulares) y la conexión con la Marvel API.
